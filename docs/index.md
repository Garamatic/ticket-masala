## Key Features

- **AI Augmentation**: GERDA AI pipeline for intelligent ticket grouping, ranking, and dispatching
- **Configuration-Driven**: YAML-based rules for extensibility without code changes
- **Modular Architecture**: Clear separation of concerns with reusable patterns
- **Multi-Domain Support**: Single platform serving IT, HR, government, and custom workflows

---

## Technology Stack

- **Backend**: .NET 10 (ASP.NET Core MVC)
- **Frontend**: Razor Views with HTMX
- **Database**: EF Core with SQLite (WAL Mode)
- **AI/ML**: ML.NET (In-Process)
- **Deployment**: Docker & Fly.io

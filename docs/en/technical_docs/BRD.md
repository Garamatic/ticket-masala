# Business Requirements Document (BRD)
**Project Name:** Ticket Masala
**Version:** 1.0
**Date:** 2025-12-12

## 1. Executive Summary
Ticket Masala is a modern, enterprise-grade ticketing and project management system designed to streamline customer support and internal task management. It integrates AI-driven automation (GERDA) to optimize ticket dispatching, categorize work, and provide actionable insights. The system is built to support multi-tenant environments with domain-specific configurations (e.g., IT Support, Government Services, Facility Management).

## 2. Business Goals
*   **Centralize Support Operations:** Provide a single platform for internal and external ticket management.
*   **Increase Efficiency with AI:** Leverage GERDA (Generic Event Reaction & Dispatch Agent) to automate assignment, prioritization, and categorization of tickets.
*   **Enhance Visibility:** Offer diverse dashboards for agents, managers, and customers to track progress and performance.
*   **Adaptability:** Ensure the system can be configured for various business domains without code changes.

## 3. Scope
### In-Scope
*   Ticket creation, management, and tracking lifecycle.
*   Project and task hierarchy management.
*   Automated dispatching and simplified "Batch Assign" workflows for managers.
*   Customer portal for ticket submission and status checking.
*   Agent portal for task execution and time tracking.
*   Quality Assurance module for reviewing ticket resolution.

### Out-of-Scope
*   Real-time chat support (future phase).
*   Integrated billing and invoicing (handled by external ERPs like SAP/Odoo).

## 4. Stakeholders
*   **Customers:** Submit tickets and require timely resolution.
*   **Agents:** Resolve tickets and need efficient tools to manage workload.
*   **Managers:** Oversee team performance, assign tickets, and viewing reports.
*   **Administrators:** Configure system settings, domains, and users.

## 5. Success Metrics
*   Reduction in average ticket resolution time.
*   Decrease in manual assignment overhead for managers.
*   High accuracy in AI-predicted tags and priority scores.
*   System uptime and responsiveness.

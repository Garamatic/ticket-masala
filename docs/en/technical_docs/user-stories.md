# User Stories

**Project:** Ticket Masala  
**System:** AI-Augmented Workflow Engine

---

## Persona: The Customer

| ID | As a... | I want to... | So that... |
| :--- | :--- | :--- | :--- |
| **US-101** | Customer | Submit a ticket via the Portal | I can report an issue without calling support. |
| **US-102** | Customer | View a live progress bar | I have realistic expectations of resolution time. |
| **US-103** | Customer | Receive "PII-Detected" warnings | I am alerted if I accidentally share sensitive data. |

---

## Persona: The Support Agent

| ID | As an... | I want to... | So that... |
| :--- | :--- | :--- | :--- |
| **US-201** | Agent | See an AI-generated summary | I don't have to read 50 legacy email chains. |
| **US-202** | Agent | Access "Twitter-Style" snippets | I can find solutions in seconds using #hashtags. |
| **US-203** | Agent | See my current workload rank | I can focus on the most critical tasks first. |

---

## Persona: The Team Manager

| ID | As a... | I want to... | So that... |
| :--- | :--- | :--- | :--- |
| **US-301** | Manager | Batch-assign the morning backlog | I can distribute work in minutes, not hours. |
| **US-302** | Manager | See "Why" AI made a suggestion | I can trust the automation and override if needed. |
| **US-303** | Manager | Enforce a monthly AI budget | We don't get surprise bills from cloud providers. |

---

## Persona: The System Admin

| ID | As an... | I want to... | So that... |
| :--- | :--- | :--- | :--- |
| **US-401** | Admin | Edit `masala_domains.yaml` | I can change the business logic without a code deploy. |
| **US-402** | Admin | Rotate API Keys for Gatekeeper | The system remains secure against unauthorized ingestion. |
